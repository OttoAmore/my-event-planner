# My Event Planner

A **React + Vite** web application for managing personal or professional events. Users can register, log in, and organize events with ease.

## Features

- User registration and login (with input validation)  
- Add, update, and delete events  
- Dashboard showing upcoming events  
- Help section with guidance for using the app  
- Responsive design for desktop and mobile  
- State management with React Context API
